Insertion Sort in C

Description

This project implements the Insertion Sort algorithm using the C programming language.
Insertion Sort is a simple sorting technique that works by building a sorted array one element at a time. It is commonly used for learning basic algorithm concepts.

The algorithm picks one element at a time and inserts it into its correct position in the already sorted portion of the array.

How Insertion Sort Works
	1.	Start from the second element of the array.
	2.	Compare the current element with the previous elements.
	3.	Shift elements that are greater than the current element to the right.
	4.	Insert the current element in its correct position.
	5.	Repeat until the entire array is Example

sorted Array:11 12 22 25 34 64 90

Code Compilation

Compile the program using a C compiler:gcc insertion_sort.c -o insertion_sort
Run the program:./insertion_sort
